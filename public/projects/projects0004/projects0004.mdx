# API RESTful Node.js

API robusta e escalável desenvolvida com Node.js e Express, incluindo autenticação, documentação e testes.

## 🌟 Características

- **Arquitetura RESTful** - Endpoints bem estruturados
- **Autenticação JWT** - Sistema seguro de autenticação
- **Documentação Swagger** - Documentação interativa
- **Testes Automatizados** - Cobertura completa de testes
- **Validação de Dados** - Validação robusta com Joi

## 🛠️ Stack Tecnológica

- **Node.js** - Runtime JavaScript
- **Express.js** - Framework web
- **MongoDB** - Banco de dados NoSQL
- **Mongoose** - ODM para MongoDB
- **JWT** - JSON Web Tokens
- **Swagger** - Documentação da API
- **Jest** - Framework de testes

## 📚 Endpoints Principais

### Autenticação
- `POST /auth/register` - Registro de usuário
- `POST /auth/login` - Login de usuário
- `POST /auth/refresh` - Renovar token

### Usuários
- `GET /users` - Listar usuários
- `GET /users/:id` - Obter usuário
- `PUT /users/:id` - Atualizar usuário
- `DELETE /users/:id` - Remover usuário

### Produtos
- `GET /products` - Listar produtos
- `POST /products` - Criar produto
- `PUT /products/:id` - Atualizar produto
- `DELETE /products/:id` - Remover produto

## 🔒 Segurança

- Autenticação JWT
