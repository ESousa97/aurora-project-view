# Bot Discord Multifuncional

Bot Discord completo com múltiplas funcionalidades, desde moderação até entretenimento e utilidades.

## 🤖 Características Principais

- **Moderação Automática** - Sistema anti-spam e filtros
- **Player de Música** - Reprodução de áudio de alta qualidade
- **Sistema de Níveis** - XP e ranking dos membros
- **Comandos Úteis** - Clima, tradução, calculadora
- **Games & Fun** - Mini-jogos e entretenimento

## 🛠️ Stack Tecnológica

- **Python 3.9** - Linguagem principal
- **Discord.py** - Library oficial Discord
- **SQLite** - Banco de dados local
- **aiohttp** - Cliente HTTP assíncrono
- **youtube-dl** - Download de áudio
- **Pillow** - Processamento de imagens

## ⚙️ Funcionalidades

### 🛡️ Moderação
- `!kick @user` - Expulsar usuário
- `!ban @user` - Banir usuário
- `!mute @user` - Silenciar usuário
- `!clear 10` - Limpar mensagens
- `!warn @user` - Sistema de advertências

### 🎵 Música
- `!play música` - Tocar música
- `!pause` - Pausar reprodução
- `!skip` - Pular música
- `!queue` - Ver fila de reprodução
- `!volume 50` - Ajustar volume

### 🎮 Games & Fun
- `!trivia` - Quiz de perguntas
- `!8ball pergunta` - Bola 8 mágica
- `!dice` - Rolar dados
- `!coinflip` - Cara ou coroa
- `!meme` - Memes aleatórios

### 📊 Sistema de Níveis
- `!rank` - Ver seu ranking
- `!leaderboard` - Top 10 usuários
- `!profile @user` - Perfil detalhado
- Sistema automático de XP

### 🌐 Utilidades
- `!weather cidade` - Previsão do tempo
- `!translate pt texto` - Traduzir texto
- `!calc 2+2` - Calculadora
- `!covid país` - Dados COVID-19

