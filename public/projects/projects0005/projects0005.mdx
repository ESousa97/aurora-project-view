# App Mobile Flutter

Aplicativo móvel multiplataforma desenvolvido com Flutter, oferecendo experiência nativa em iOS e Android.

## 📱 Características

- **Multiplataforma** - Um código para iOS e Android
- **Performance Nativa** - Interface fluida e responsiva
- **Material Design** - Seguindo diretrizes do Google
- **Offline First** - Funciona sem conexão
- **Push Notifications** - Notificações em tempo real

## 🛠️ Tecnologias

- **Flutter 3.0** - Framework UI
- **Dart** - Linguagem de programação
- **Firebase** - Backend as a Service
- **Provider** - Gerenciamento de estado
- **Dio** - Cliente HTTP
- **Hive** - Banco local
- **Firebase Messaging** - Push notifications

## 🎨 Interface

### Telas Principais
- **Splash Screen** - Tela de carregamento
- **Onboarding** - Introdução ao app
- **Login/Registro** - Autenticação
- **Home** - Tela principal
- **Perfil** - Dados do usuário
- **Configurações** - Preferências

### Componentes
- Bottom Navigation
- Custom App Bar
- Cards personalizados
- Loading indicators
- Modals e dialogs

## 🔥 Integração Firebase

- **Authentication** - Login social e email
- **Firestore** - Banco de dados
- **Storage** - Armazenamento de arquivos
- **Messaging** - Push notifications
- **Analytics** - Métricas de uso
- **Crashlytics** - Relatórios de crash

## 📊 Funcionalidades

### Core Features
- Sistema de autenticação
- Lista de itens com busca
- Perfil do usuário
- Notificações push
- Modo offline

### Recursos Avançados
- Geolocalização
- Câmera e galeria
- Compartilhamento
- Deep linking
- Biometria

## 🎯 Gerenciamento de Estado

Utilizando Provider para:
- Estado de autenticação
- Dados do usuário
- Lista de itens
- Configurações do app
